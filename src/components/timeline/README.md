# Timeline 时间轴

## API

|  属性   | 说明  | 类型 |  默认值 |
|  ----  | ----  | ---- | ---  |
|  color  | 圆圈颜色，可选值为`blue`、`red`、`green`、`gray`，或自定义色值  | Array [ string ] | blue  |
|  pending  | 指定是否最后一个节点为幽灵节点  | Boolean | false  |

## slot

|  名称   | 说明  |
|  dot  | 自定义时间轴点  |
|  timelineItem  | 时间轴描述内容  |
