# Result 结果

## API

|  属性   | 说明  | 类型 |  默认值 |
|  ----  | ----  | ---- | ---  |
| title | 标题文字 | String | - |
| subTitle | 副标题文字 | String | - |
| status | 结果的状态，决定图标和颜色，`info`、`success`、`warning`、`error`、`403`、`404`、`500` | String / Number | info |
| icon | 自定义 icon | String | - |

## slot

|  名称   | 说明  |
|  ----  | ----  |
| footer | 按钮栏和链接栏 |
| content | result详细描述内容 |
