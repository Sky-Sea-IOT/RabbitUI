# BackTop回到顶部

## API

|  属性   | 说明  | 类型 |  默认值 |
|  ----  | ----  | ---- | ---  |
| height | 页面滚动高度达到该值时才显示组件 | Number | 400 |
| bottom | 组件距离底部的距离 | Number | 50 |
| right | 组件距离右部的距离 | Number | 100 |
| duration | 回到顶部所需时间（ms） | Number | 450 |
| onClick | 点击按钮的回调函数 | Function | - |

## Slot

|  名称   | 说明 |
|  ----  | ----  |
| content | 自定义回到顶部按钮的样式和内容 |
