# Spin 加载中

## API

Spin 实例

使用内置的 Spin 方法可以全局加载

- `rabbit.Spin.show(config)`
- `rabbit.Spin.hide()`

|  属性   | 说明  | 类型 |  默认值 |
|  ----  | ----  | ---- | ---  |
| size | Spin尺寸，可选值为large和small或者不设置 | String | - |
| text | 当此项有内容时会以圆环旋转外观的加载中图标显示 | String | - |
| color | 外观为点状时可设置颜色，可选值为 primary、info、success、warning、error | String | primary |
